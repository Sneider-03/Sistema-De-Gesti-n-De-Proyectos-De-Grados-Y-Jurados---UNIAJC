"Sistema de Gestioon de Proyectos de Grado de la UNIAJC

En la UNIAJC, los estudiantes deben desarrollar un proyecto de grado como requisito para optar al título profesional. 
Este proceso implica múltiples actividades como el registro del proyecto, la revisión del anteproyecto, la entrega de documentos, 
la asignación de directores y jurados, las evaluaciones parciales y finales, y la emisión de actas oficiales.
Actualmente, gran parte de estas tareas se llevan a cabo de manera manual o utilizando herramientas no integradas,
lo que genera dificultades como pérdida de información, falta de seguimiento, escasa trazabilidad y errores en la asignación 
de actores o fechas. Esto justifica la necesidad de un sistema centralizado que permita gestionar todo el ciclo de vida del proyecto de manera eficiente.
